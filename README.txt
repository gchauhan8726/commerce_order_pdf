Commerce Order PDF -  module provides the facility to print the invoice of
commerce order in pdf format.

This module integrates the Dompdf library into your Drupal site to print the
invoice of commerce order.


REQUIREMENTS
============
- commerce_order


FEATURES
========

Commerce Order PDF provides a button to print the order invoice in the form of
PDF. When you will click on the button print invoice at /commerce/orders page
it will print the order invoice. Now you can download the order invoice.


INSTALLATION
============
1- download the Commerce Order PDF module from:
   https://www.drupal.org/project/commerce_order_pdf
2- Copy the Commerce Order PDF folder to your modules directory.
3- Now run command 'composer install' inside the module to install the Dompdf.
4- You can also download Dompdf library from https://github.com/dompdf/dompdf.
5- Copy the downloaded content that is Dompdf folder in the root libraries
   folder (/libraries).
6- Go to admin/modules and enable the module.
